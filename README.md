Krátký návod pro spuštění PyGame ukázky

Požadavky
- Python 3.8+
- nainstalovaný pygame (viz níže)

Instalace závislostí (PowerShell):

```
python -m pip install -r requirements.txt
```

Spuštění (PowerShell):

```
python .\main.py
```

Chování programu
- Stiskem Esc nebo zavřením okna program ukončíte.
- Pohyb čtverce: šipky nebo WASD.
